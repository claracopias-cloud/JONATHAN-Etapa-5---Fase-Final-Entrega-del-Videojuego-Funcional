# JONATHAN-Etapa-5---Fase-Final-Entrega-del-Videojuego-Funcional
To create a functional prototype of a directly executable video game, we will use Unity with a style where the player controls two characters: the enemy and the main character.
A functional prototype is the basic, testable version. Its main objective is to validate the core mechanics, such as attack, movement, defeat, jump, and run.
